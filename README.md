# freeSCHEMA To-Do App

A To-Do application built using the **freeSCHEMA** framework. This project demonstrates how to handle CRUD operations (Create, Read, Update, Delete) with a centralized Semantic Concept Connection System.

---

## 🚀 Getting Started

Follow these steps to get the project running on your local machine.

### 1. Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed.

### 2. Installation
Clone the repository and install the necessary dependencies:

```bash
git clone [https://github.com/Umm210/freeSchema_todo.git](https://github.com/Umm210/freeSchema_todo.git)
cd freeSchema_todo
npm install
```
### 3. Running the Project
Start the development server:

```bash
npm run dev
```

Once the server starts, open your browser to the URL shown in your terminal (usually http://localhost:5173 or http://localhost:3000).

### 4.How to Use

**Add Tasks:** Use the input field at the top to type a task and click Save Task.



**Edit:** Click the Edit button next to any task. This will populate the top input field with the existing text, allowing you to update it.



**Delete:** Click the red Delete button to permanently remove a task from the database.
