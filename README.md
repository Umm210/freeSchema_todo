# freeSCHEMA To-Do App

A To-Do application built using the **freeSCHEMA** framework. This project demonstrates how to handle CRUD operations (Create, Read, Update, Delete) with a centralized Semantic Concept Connection System.

---

## 🚀 Getting Started

Follow these steps to get the project running on your local machine.

### 1. Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed.

### 2. Installation
Clone the repository and install the necessary dependencies:

```bash
git clone [https://github.com/Umm210/freeSchema_todo.git](https://github.com/Umm210/freeSchema_todo.git)
cd freeSchema_todo
npm install```
